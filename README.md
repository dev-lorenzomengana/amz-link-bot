# 🤖 @personal_affiliate_amazon_bot

Bot Telegram che converte i link Amazon in link affiliati personalizzati. Ogni utente utilizza il proprio tag Amazon e riceve il link pronto da condividere.

## ✅ Il bot è online
Telegram: @personal_affiliate_amazon_bot

## Cosa fa
- Converte link Amazon con il tuo tag affiliato
- Supporta link normali e corti: `amzn.to`, `a.co`, `amzn.eu`
- Funziona inline in qualsiasi chat Telegram
- Ogni utente ha il proprio tag affiliato

## Comandi principali
- `/start` — avvia il bot
- `/settag <tag>` — imposta il tuo tag
- `/help` — aiuto
- `/export` — esporta i tuoi dati
- `/deletedata` — elimina i tuoi dati

## Uso rapido
1. Apri Telegram
2. Avvia @personal_affiliate_amazon_bot
3. Invia `/settag nome-tag`
4. Invia un link Amazon

## Inline
Usa il bot in qualsiasi chat con:

`@personal_affiliate_amazon_bot https://amazon.it/dp/B08N5WRWNW`

Il bot invia direttamente il link affiliato.
